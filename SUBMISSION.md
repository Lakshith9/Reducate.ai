# Submission Notes

Use this file to finish the internship assignment submission after the project is uploaded and hosted.

## 1. Install Tools

Install Node.js LTS from:

https://nodejs.org/

Install Git from:

https://git-scm.com/downloads

Restart the terminal after installation.

## 2. Run And Verify

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

Check:

- Desktop view
- Mobile responsive view
- Navbar links
- Mobile menu
- Hover transitions
- Partner marquee animation
- Full page scroll

## 3. Create GitHub Repository

```bash
git init
git add .
git commit -m "Build modern JG University landing page"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/jg-university-landing.git
git push -u origin main
```

## 4. Host The Project

Recommended: Vercel

1. Go to https://vercel.com/
2. Import the GitHub repository
3. Keep the default Next.js settings
4. Deploy
5. Copy the hosted URL

## 5. Demo Video Outline

Record a 2-5 minute video covering:

1. Open the hosted site on desktop
2. Show the hero section and CTA buttons
3. Scroll through About, Programs, Campus, Testimonials, CTA, and Footer
4. Hover over cards/buttons to show interactions
5. Open mobile responsive view in browser dev tools
6. Show the mobile navigation menu
7. Scroll through the mobile page
