# JG University Landing Page

A modern, responsive landing page concept inspired by [JG University](https://jguni.in/), built with Next.js and Tailwind CSS.

## Tech Stack

- Next.js App Router
- Tailwind CSS
- TypeScript
- Lucide React icons

## Sections

- Responsive navbar with mobile menu
- Full-screen hero section
- About section
- Programs section
- Strengths and features
- Campus showcase
- Alumni testimonials
- Partner marquee
- CTA section
- Footer

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Build

```bash
npm run build
npm run start
```
